# Simple Python CLI To-Do App

A minimalist command-line interface (CLI) To-Do application built with Python. This application allows users to manage tasks by saving them directly to a local file (`todo.txt`).

* **View Tasks:** Display all current tasks with their status (done/not done).
* **Add Tasks:** Easily add new tasks to the list.
* **Remove Tasks:** Delete tasks using their corresponding number.
* **Modify Tasks:**
    * Toggle the status of a task (mark as done or undone).
    * Change the description of an existing task.
* **Local Persistence:** All tasks are saved to and loaded from a file named `todo.txt` in the same directory.
